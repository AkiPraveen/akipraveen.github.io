Cross-platform chemistry reference data application utilizing client-side algorithms in conjunction with web scraping from trusted sources in order to compile resources optimized for student accessibility. Release sponsored by the Ideated incubator.



DETAILS:


This project was created at HackingEDU 2015. The demo version of the project can be found in the Github repository linked above. My contributions to this project are as follows:

  • Created basic website and structure using HTML5, CSS3, and JavaScript
  • Created a smooth, responsive, and functional UI that worked on laptops, desktops, and smartphones using Bootstrap and JQuery
  • Implemented Noodle.js to web-scrape reference data from various websites, implemented PDF.js to scrape data from chemical MSDS sheet PDF files.
  • Wrote JavaScript algorithm to calculate molar mass
